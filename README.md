# Deadline Application Tracker

A Flutter mobile application that helps students track academic deadlines and stay organized, with a clean, intuitive interface built for everyday coursework management.

## Features

* Add and manage academic deadlines
* Organize tasks by course or category
* Get a clear overview of upcoming due dates

## Techniques Used

* Cross-Platform Mobile Development
* State Management
* Mobile UI/UX Design
* Local Data Persistence

## Tools

* Flutter
* Dart

## How it works

The app is built entirely in Flutter, allowing it to run across mobile, desktop, and web from a single codebase. Students can add assignments and deadlines through a simple interface, which organizes and displays tasks so upcoming due dates are easy to track at a glance.
